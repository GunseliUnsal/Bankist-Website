# Bankist-Website
Bankist website is a project built with dynamic effects using JavaScript.
I made this project while learning Advanced DOM and Events from Jonas Schmedtmann's course.

## Live Demo
**You Can See The Project Live On:** [Here](https://gunseliunsal.github.io/Guess-my-Number/)

Features
* Modal windows 
* Smooth scrolling 
* Tab functionality 
* Navigation fade animations 
* Sticky navigation 
* Revealing sections on scroll 
* Lazy images loading 
* Slider functionality
